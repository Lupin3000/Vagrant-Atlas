Example

Create environment from template:

# create new project
$ mkdir ~/CentOS7 && cd ~/CentOS7

# download Vagrantfile template via curl
$ curl -o Vagrantfile http://atlas.com/vagrant/templates/Vagrantfile_CentOS7

# edit Vagrantfile
$ vim Vagrantfile

# start environment
$ vagrant up

... do your great work ...

# destroy environment
$ vagrant destroy -f
